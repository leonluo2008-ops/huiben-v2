## 2026-04-18 15:35 规则更新：标题+旁白大师技巧

### 标题公式
- 短小（3-8字）+ 画面感 + 暗示故事核心（不概括）
- 三角度出标题：核心意象 / 情感角度 / 叙事钩子
- 禁止：概括故事、抽象概念、过长

### 简介公式
- 1-2句 + 感官画面 + 情感钩子 + 激发好奇
- 禁止：概括故事、空泛形容词、过长句子

### 旁白公式（Oliver Jeffers）
- 12句结构：开头(1-2) → 冲突(3-4) → 发展(3-4) → 高潮(1-2) → 结局(1-2)
- 每句视觉化，用感官细节替代抽象描述
- 情感克制但深沉，不说教
- 禁止：说教、过度解释、场景跳脱、重复画面
